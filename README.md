## プライバシー

すべてのツールはクライアントサイド(ブラウザ内)で完結します。データがサーバーに送信されることはありません。

## 技術スタック

- **フロントエンド**: React 18 + TypeScript
- **ビルドツール**: Vite 6
- **スタイリング**: Tailwind CSS 3.4 + shadcn/ui (Radix UI)
- **ホスティング**: Cloudflare Pages
- **ランタイム**: Bun
- **Linter**: Biome

## 開発

```bash
# 依存関係のインストール
bun install

# 任意のアプリの開発サーバー起動
cd apps/<app-name>
bun run dev

# 全体lint
bun run lint

# テスト
bun test

# Storybook
bun run storybook
```

詳しくは [開発者向けドキュメント](./__docs__/README.md) をご覧ください。
